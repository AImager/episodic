配置 GitHub 同步，导出的对话会直接 commit 到你的知识库仓库。

如何生成最小权限 Token

  1. 打开 GitHub → Fine-grained tokens
  2. 「Generate new token」，Repository access 选 Only select repositories，只勾选你的笔记仓库
  3. Permissions → Repository permissions → Contents 设为 Read and write
  4. 生成后复制 github_pat_... 粘贴到上方，保存即可
Token 仅存于本浏览器的 chrome.storage.local，不会上传到任何第三方。
